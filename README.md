# 🌟 AIFriends

AIFriends 是一个基于大语言模型（LLM）的全栈虚拟陪伴平台。在这里，你可以零代码创建、定制并分享专属的虚拟女友、男友或知心好友。通过引入先进的语音处理技术与 RAG 架构，AIFriends 致力于打破次元壁，为你提供最自然、沉浸的情感陪伴与智能交互体验。



## ✨ 核心特性

* **🎭 深度角色定制与分享**
  * 支持创建无限量专属虚拟角色。
  * 全方位自定义设定：包括角色的核心性格、背景故事、聊天风格以及专属音色。
  * 社区生态：支持将你创造的特色角色一键分享给其他用户。

* **🎙️ 沉浸式实时语音交互**
  * **自然语音通话**：打破传统的纯文本聊天，支持与虚拟人物进行实时语音连线。
  * **语音识别 (ASR)**：精准捕捉并理解你的每一次开口。
  * **语音合成 (TTS) & 声音复刻**：不仅支持文字转语音，更能克隆并绑定特定音色，实现宛如真人的听觉体验。

* **🧠 硬核 AI 能力驱动**
  * **大模型中枢**：底层采用 LangChain 框架进行复杂的提示词编排与记忆管理。
  * **Function Calling (函数调用)**：赋予虚拟人物感知真实世界和使用外部工具的能力（如查询实时信息、执行特定任务等）。
  * **私有知识库 (RAG)**：支持为角色挂载专属文档与背景设定，让你的 AI 朋友拥有特定领域的知识储备和上下文连贯的长期记忆。
